ダウンロードありがとうございます

●TSOWeightCopy ver 0.0.2

これはなに
頂点ウェイトを鏡面コピーするツールです。

動作環境
・.NET Framework 3.5
・DirectX 9.0c

使い方
コマンドプロンプトから起動します。
  TSOWeightCopy.exe your.tso

途中で
・処理するメッシュ
・鏡面コピーの方向
を尋ねるために入力待ちになります。数値を入力してEnterキーを押します。

解析に成功すると、ファイル名に.new.tsoを付けた名前でtsoファイルを作成します。

実行例
  TSOWeightCopy.exe N004SAIL_M00.tso

  メッシュ:
  0 SkirtCHECK_SHORT
  メッシュを選択 (0-0): 0  ### '0' を入力
  頂点数:2296
  min:[ -2.39507 0 -2.59799 ]
  max:[ 2.39507 12.5935 1.34922 ]
  同一視頂点数:670
  
  方向:
  0 左から右
  1 右から左
  方向を選択 (0-1): 0  ### '0' を入力
  ...
  Save File: N004SAIL_M00.new.tso

注意
・必ずtsoファイルをバックアップしてください。


設定ファイル
・flipnodes.txt：左右を入れ替えるボーンのリストです。


ver 0.0.1 からの変更点：
・ボーン参照リストにない場合は追加
