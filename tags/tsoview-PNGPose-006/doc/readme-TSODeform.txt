●TSODeform ver 0.0.3

これはなに
tsoをtmoで変形して保存します。

動作環境
・.NET Framework 3.5
・DirectX 9.0c

使い方
コマンドプロンプトから起動します。
  TSODeform.exe your.tso your.tmo Figure1.txt

変形に成功するとout.tsoを作成します。

Figure1.txt：スライダの位置（省略可）
シーンデータをPNGPoseで展開すると出力されるファイルです。
値は float型で 0.0 .. 1.0
    0: 姉妹
    1: うで
    2: あし
    3: 胴まわり
    4: おっぱい
    5: つり目たれ目
    6: やわらか

作成したtsoの使い道：
・tso2mqoでmqoに変換する：ポーズを反映して展開されます。
・tso2mqoで参照tsoに使う：ポーズを反映したボーンになります。

ver 0.0.2 からの変更点：
・スライダ値を指定しない場合は補正を行わない

ver 0.0.1 からの変更点：
・スライダ値を読み込める
